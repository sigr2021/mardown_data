# Ecole thématique Sciences de l'information géographique reproductibles SIGR

[Ecole thématique CNRS](https://sigr2020.sciencesconf.org/resource/page/id/2), 27 juin au 2 juillet 2021 à Saint-Pierre d'Oléron. 

Ce document R Markdown présente les données mobilisées lors de l'atelier [**Produire des documents reproductibles avec R Markdown**](https://sigr2021.github.io/markdown/), animé par Timothée Giraud, Ronan Ysebaert et Nicolas Lambert (UMS RIATE).

Les sources des données et la préparation des données sont présentées, et quatre visualisations cartographiques permettent de qualifier les environs géographiques des centres de vacance CNRS (démographie, pollution lumineuse, équipements touristiques).

[Accéder au document](https://sigr2021.github.io/mardown_data/)
